# NeteaseCloudMusic

#### 项目介绍
Music Player for Windows

#### 软件架构
 1. Node.js
 2. Electron.js
 3. Vue.js全家桶
 4. Webpack
 5. Sass

#### 环境要求

 需要 NodeJS 6.0+ 环境


#### 安装

1. Node 为必须安装环境
2. 安装依赖：npm install
3. 运行：npm run dev
4. 打包（根据不同的平台）：npm run build:win32 / build:linux / build:mas / build:web

- 搜索
- 每日推荐

- 喜欢的音乐
- 个人歌单
